Laboratório 1

Escreva um código que solicite ao usuário entrar com dois números e você deve imprimir a soma de todos os números pares entre esses dois números(incluindo os números digitados se forem pares).

* Faça o tratamento para garantir que o usuário vai sempre digitar um número.
* Faça o tratamento necessário para que o segundo número seja menor que o primeiro.
* Use a função prompt para capturar os dados do usuário.
* Use o console.log() para mostrar o resultado.
